# Licensing Guide

## namgun-workspace 라이선스

namgun-workspace는 **AGPL-3.0** (GNU Affero General Public License v3.0)으로 배포됩니다.

### AGPL-3.0 핵심 조건

1. **소스 공개**: 수정된 버전을 배포하거나 네트워크 서비스로 제공할 때 소스 코드를 공개해야 합니다.
2. **동일 라이선스**: 파생 작업물도 AGPL-3.0으로 배포해야 합니다.
3. **네트워크 조항**: 웹 서비스로 제공하는 것도 "배포"에 해당합니다.

### 자유롭게 할 수 있는 것

- 자체 서버에서 팀/조직을 위해 운영
- 소스 코드 수정 및 커스터마이징
- 수정 버전을 AGPL-3.0으로 공개 배포

### 주의사항

- 수정된 버전을 네트워크 서비스로 제공할 경우 소스 공개 의무
- 상용 라이선스가 필요한 경우 별도 문의
